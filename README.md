# KGG Producciones

Sitio estático para KGG Producciones, productora audiovisual de Buenos Aires enfocada en marcas y eventos.

- Reels para marcas
- Coberturas de eventos
- Foto y video de producto
- Edición de contenido
- Contenido digital mensual
- Videos institucionales

## Cómo probar

Abrir `index.html` en el navegador o servir la carpeta con un servidor estático local.

Revisar:

- Hero en desktop y mobile, sin scroll horizontal.
- Botón "Pedir propuesta" y formulario de contacto.
- Apertura de WhatsApp con el mensaje precargado.
- Menú mobile, filtros de trabajos y botones flotantes.

## Mantenimiento

- Actualizar teléfono, email y redes en `index.html`.
- Reemplazar trabajos por links reales cuando haya piezas publicadas.
- Mantener optimizada la imagen principal: el sitio usa `assets/hero-visual-creator-optimized.jpg` y conserva el PNG como fallback.

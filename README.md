# Landing de contenido visual

Proyecto web estático listo para abrir en Visual Studio o Visual Studio Code.

## Archivos

- `index.html`: estructura de la página.
- `styles.css`: diseño responsive, colores, grillas y animaciones.
- `script.js`: menú mobile, filtros del portfolio, animaciones y formulario simulado.
- `assets/hero-visual-creator.png`: imagen principal generada para el sitio.

## Cómo abrirlo

1. Abrí Visual Studio o Visual Studio Code.
2. Elegí **Open Folder / Abrir carpeta**.
3. Seleccioná esta carpeta: `C:\Users\Clau\OneDrive\Documents\New project`.
4. Abrí `index.html` en el navegador.

Si usás Visual Studio Code, también podés usar la extensión **Live Server** para verlo con recarga automática.

## Personalización rápida

- Cambiá `Nombre` por tu nombre o marca en `index.html`.
- Reemplazá `@tu.usuario`, `hola@tudominio.com` y el número de WhatsApp.
- Sustituí `assets/hero-visual-creator.png` por fotos propias si ya tenés portfolio real.
